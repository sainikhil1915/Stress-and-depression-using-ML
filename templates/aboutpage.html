About the Project

This web application is designed to detect mental distress, such as stress or depression, using machine learning models. The detection is based on either user-generated social media content (tweets) or responses to standardized depression assessment questions.

Detection Methods

  • Tweet-based Detection: Uses Natural Language Processing (NLP) on user tweets to detect stress/depression.
  • Depression Questionnaire-based Detection: Evaluates responses to 10 clinically relevant questions using ML models.

Technologies Used

  • Python, Flask for Web Development
  • Scikit-learn for Machine Learning
  • TF-IDF for Feature Extraction
  • Joblib for Model Serialization
  • SQLite for User and Prediction Data Storage

Model Accuracies

On Depression Questionnaire Dataset:
  • Random Forest: 97%
  • Support Vector Machine (SVM): 96%
  • Multilayer Perceptron (MLP): 95%
  • Naive Bayes: 93%
  • Logistic Regression: 95%
  • Gradient Boosting: 96%
  • Decision Tree: 95%
On Tweet Dataset:
  • Random Forest: 97%
  • Support Vector Machine (SVM): 96%
  • Multilayer Perceptron (MLP): 95%
  • Naive Bayes: 90%
  • Logistic Regression: 95%
  • Gradient Boosting: 96%
  • Decision Tree: 94%

Users can register, log in, and test the system with their tweet or questionnaire input. An admin panel stores all results securely for future analysis.
